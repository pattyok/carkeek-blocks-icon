Block that allows user to search for a font-awesome icons.

== Description ==

Place an icon block on the page

== Installation ==


1. Download the latest release of the plugin
1. Upload the plugin files to the `/wp-content/plugins/carkeek-blocks-icon` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Subsequent updates are pushed through github. [Use the Git Updater Plugin, 'https://git-updater.com/'] to manage updates.

== About Font Awesome ==
This plugin uses Font Awesome v6. To use, Font Awesome needs to be installed on the site and available within the site admin.

To update the font pack in the plugin. Go to Font Awesome and download the web font, use the json files in the download.
